# Revolución Francesa: Arde París 🎮

Sitio web oficial del videojuego conceptual **Revolución Francesa: Arde París**.  
Desarrollado por **Liberation Studios** — Clasificación: M (Maduro).

## 🚀 Cómo ejecutar

1. Clona o descarga este repositorio
2. Abre `index.html` en tu navegador — **no se necesita servidor**
3. La carpeta `music/` debe estar en la misma ubicación que `index.html`

## 📁 Estructura del proyecto

```
/
├── index.html          ← Página principal (todo el sitio en un solo archivo)
├── README.md           ← Este archivo
└── music/              ← Canciones de la banda sonora original
    ├── README.md       ← Instrucciones sobre los archivos de audio
    ├── 01_Maximilien_Robespierre_La_Virtud_y_el_Terror.mp3
    ├── 02_Luis_XVI_El_Peso_de_la_Corona.mp3
    ├── 03_María_Antonieta_Cristal_Roto_en_Versalles.mp3
    ├── 04_Marqués_de_La_Fayette_Entre_Dos_Mundos.mp3
    ├── 05_Georges-Jacques_Danton_La_Voz_del_Trueno.mp3
    └── 06_Napoleón_Bonaparte_El_Vuelo_del_Águila.mp3
```

## 📱 Compatibilidad

- ✅ PC / Mac (Chrome, Firefox, Safari, Edge)
- ✅ Tablet (iPad, Android)
- ✅ Móvil (iOS Safari, Android Chrome)

## 🌐 Publicar en GitHub Pages

1. Sube este repositorio a GitHub (público)
2. Ve a **Settings → Pages → Deploy from branch → main → / (root)**
3. En unos minutos tendrás tu URL: `https://tuusuario.github.io/nombre-repo`

## ℹ️ Notas

- El reproductor de audio requiere que los archivos `.mp3` estén en la carpeta `music/`
- Si los archivos no están, el reproductor mostrará un mensaje de error claro
- El sitio funciona completamente offline una vez descargado con los archivos de audio
